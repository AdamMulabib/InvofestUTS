# InvoFest UAS

Project website management event menggunakan React TypeScript, Express.js, Prisma ORM, dan PostgreSQL.

## Tech Stack
- Frontend: React TS + Tailwind CSS
- Backend: Express TS + Prisma ORM
- Database: PostgreSQL
- Frontend Deployment: Vercel
- Backend Deployment: Railway

## Repository Structure
/frontend -> React Frontend
/backend -> Express Backend
